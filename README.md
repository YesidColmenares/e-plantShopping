# e-plantShopping 🌿

## Proyecto: Paradise Nursery - Shopping Cart App

Este repositorio **e-plantShopping** contiene una aplicación web desarrollada con React y Redux que simula un carrito de compras para una tienda de plantas de interior llamada *Paradise Nursery*.

## 📌 Descripción

La aplicación permite a los usuarios:

- Visualizar una página de inicio con información sobre la tienda.
- Navegar a una página de listado de productos organizada por categorías.
- Agregar plantas al carrito de compras.
- Aumentar, disminuir o eliminar productos del carrito.
- Visualizar el total dinámicamente actualizado.
- Continuar comprando o proceder al pago.

## 🚀 Tecnologías utilizadas

- React
- Redux Toolkit
- React Hooks (useState, useEffect)
- JavaScript (ES6)
- CSS

## 🎯 Funcionalidades implementadas

- Componentes funcionales con composición.
- Gestión global del estado con Redux.
- Renderización dinámica utilizando map().
- Manejo de eventos y renderización condicional.
- Cálculo automático del total del carrito.
